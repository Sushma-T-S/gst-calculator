# GST Calculator

A simple GST Calculator built using Next.js and Tailwind CSS.

## Features

- Calculate GST amount instantly
- Calculate total amount including GST
- Simple and responsive user interface
- Built using Next.js 16

## Technologies Used

- Next.js
- React
- TypeScript
- Tailwind CSS
